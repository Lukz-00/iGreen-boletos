Configurar Google Sheets

Configure a planilha central onde os dados ficam armazenados e são compartilhados com todos os colaboradores

📋 Como configurar — passo a passo

Acesse sheets.google.com e crie uma nova planilha. Dê o nome "Central Boletos iGreen".

Na planilha, vá em Extensões → Apps Script e cole o código do Web App que está logo abaixo. Salve e clique em Implantar → Novo deployment → App da Web.

Em "Quem tem acesso", selecione "Qualquer pessoa". Clique em Implantar e copie a URL do Web App gerada.

Cole a URL abaixo no campo URL do Web App e clique em Salvar. Pronto — todos os colaboradores verão os dados ao clicar em Sincronizar.

Para compartilhar com colaboradores: publique este arquivo HTML no GitHub Pages e envie o link. Nenhuma instalação necessária.

⚙ URL do Web App (Google Apps Script)

Cole aqui a URL do seu Web App
A URL fica salva localmente nesta máquina. Todos os colaboradores que usarem o link do GitHub Pages precisarão que a gestora configure a URL no primeiro acesso da máquina deles — ou você pode fixar a URL diretamente no código.

Dica rápida: após salvar, clique em ↻ Sincronizar na barra superior para testar a conexão.
